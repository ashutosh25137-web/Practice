# Arranging Clothes in Almirah

1. Take input as number of shelves n ( mention in problem ) in the almirah.
2. Collect all clothes from the almirah.
3. Identify clothes into separate categories - 
                                               -Dailywear
                                               -Formalwear
                                               -Traditionalwear
                                               -Sportswear
4. Confirm number of shelves available in almirah.
5. Confirm each shelves for each categories
6. Folds all categories clothes neatly to save space.
7. Confirm shelvess based of frequency of use:
     - Keep most uses cloths on middle shelves for easy access
     - Keep less uses cloths on top shelves
     
8. Keep similar clothes together to reduce search time.
9. Label the each shelf according to clothing category
10. Confirm no empty space is wasted.
11. Close the almirah after confirming proper arrangement
